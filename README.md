# api-to-do-list-spring
Api rest de todo list (lista de tarefas) com Swagger e CRUD completo em java e Spring boot.

<h3><li>Tecnologias usadas</li></h3>
    Java 11<br>
    Spring Boot<br>
    Swagger<br>
    JPA<br>
    Hibernate<br>
    H2 Database<br>
